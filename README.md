# 鳴潮賽事專用 Ban/Pick 系統 (Wuthering Waves BP Tool)

🔗 **系統連結：** [https://quaso-373.github.io/wuwa-bp-tool](https://quaso-373.github.io/wuwa-bp-tool)

專為《鳴潮 Wuthering Waves》社群賽事打造的網頁版實時 Ban/Pick (禁用/選擇) 工具。提供連線同步，以及完善的裁判控制功能，讓線上賽事也能擁有職業電競級的 BP 體驗

---

## ✨ 核心特色功能

* **⚡ 實時零延遲同步：** 基於 WebRTC 技術，房主與選手的畫面、選角、倒數計時皆為全網實時同步
* **🎭 沉浸式視聽體驗：** * 支援角色專屬語音播放與電影級低頻鎖定音效
  * 破壞階段（Ban）專屬紅色碎裂打擊特效
  * 3D 視差閃卡滑動效果與精緻的半透明毛玻璃 UI
* **⚖️ 完善的賽事管理：** 裁判擁有最高權限，可隨時操作「暫停/開啟計時器」、「Undo 上一步」以及「無縫軟重置（不踢人直接開下一局）」
* **💬 內建賽事聊天室：** 選手、裁判與觀眾可即時文字交流，系統也會自動播報進房狀態
* **📋 戰報一鍵生成：** BP 結束後可一鍵複製純文字戰報，方便賽後紀錄與發布

---

## 📜 賽事 BP 規則與流程

本系統採用 **「先 Ban ➡️ 輪流 Pick ➡️ 最終 Snipe (破壞)」** 的賽制邏輯：

1. **禁用階段 (Ban Phase)：**
   * 藍隊 (Blue) 先禁用 1 名角色 ➡️ 紅隊 (Red) 禁用 1 名角色
2. **選擇階段 (Pick Phase)：**
   * 雙方交替選角，順序為：`藍 1` ➡️ `紅 2` ➡️ `藍 2` ➡️ `紅 2` ➡️ `藍 2` ➡️ `紅 2` ➡️ `藍 2` ➡️ `紅 1`
   * *(註：雙方第 3 個選取的位置為「保護位 🛡️」，將免疫後續的破壞階段)*
3. **破壞階段 (Snipe Phase)：**
   * 雙方陣容選定後，進入最終破壞階段
   * 藍隊先點擊並「破壞」紅隊陣容中的 1 名角色 ➡️ 紅隊「破壞」藍隊 1 名角色
   * 被破壞的角色將無法在該局比賽中使用（保護位除外）

## 📖 使用方式說明

進入網站後，請先輸入「自訂暱稱」，並根據您的參賽身分進行以下操作：

### 👑 裁判 / 轉播方 (房主)
1. 選擇身分（不影響建房，但建議選觀眾），點擊 **「建立房間」**
2. 系統會生成一組 **4 碼房間代碼**，請將此代碼發送給雙方選手
3. 裁判擁有最高操作權限（可代點任何隊伍的選角）
4. **下方控制列面板說明：**
   * `⏱️ 30s`：開關每回合 30 秒的倒數計時
   * `UNDO`：選錯或操作失誤時，退回上一個動作
   * `COPY REPORT`：一鍵複製當前 BP 結果
   * `RESET`：清空當前 BP 畫面並進入下一局（不會中斷選手連線）

### ⚔️ 參賽選手
1. 輸入暱稱，選擇您所屬的隊伍（**🔵 藍隊選手** 或 **🔴 紅隊選手**）
2. 輸入裁判提供的 **4 碼房間代碼**，點擊 **「加入房間」**
3. 進入大廳後等待裁判指示，**只有在輪到己方回合時，畫面上的選角操作才會生效**

### 👀 觀眾
1. 輸入暱稱，身分選擇 **「👀 觀眾」**
2. 輸入裁判提供的代碼加入房間
3. 觀眾無法進行任何畫面點擊與選角操作，但可即時觀看 BP 過程，並使用右下角「聊天室」進行討論

## 📁 專案維護說明 (給開發者)
若未來需要新增角色或語音，請將對應檔案放入以下資料夾，並確保檔名與代碼中的 `name` 或 `fileName` 完全一致：
* `/avatar/`：存放選角池使用的小頭像 (1:1 `.webp`)
* `/portrait/`：存放舞台區使用的等身立繪 (`.webp`)
* `/audio/`：存放選角時播放的專屬語音 (`.mp3`)
